
              Hipparchus 2.2 RELEASE NOTES

The Hipparchus team is pleased to announce the release of hipparchus-2.2

The Hipparchus library is a library of mathematical components. It started as a fork of Apache Commons Math.

Changes in this version include:

New features:
o issue/190:  Added {Field}TaylorMap, supporting evaluation, composition and inversion. 
o issue/200:  Added FieldMatrixDecomposer. 
o issue/191:  Added rebase to {Field}DerivativeStructure, which enables changing
              the base variables a {Field}DerivativeStructure depends on. 
o issue/199:  Added Unscented Kalman Filter for non linear estimations. Thanks to Gaetan Pierre. 
o issue/198:  Added CombinatoricsUtils.permutations. 
o             Added computation of Bell numbers. 
o issue/197:  Added RosenNumberPartitionIterator. Thanks to Axel Kramer. 
o issue/197:  Added CombinatoricsUtils.partitions. 
o issue/195:  Added FieldMultivariateFunction, CalculusFieldMultivariateFunction,
              FieldMultivariateVectorFunction, CalculusFieldMultivariateVectorFunction,
              FieldMultivariateMatrixFunction and CalculusFieldMultivariateMatrixFunction 
o issue/192:  Added integration/differentiation of {Field}DerivativeStructure. Thanks to Romain Serra. 

Fixed Bugs:
o issues/194:  Added a getPrimary method to FieldExpandableODE. 


For complete information on Hipparchus, including instructions on how to submit bug reports,
patches, or suggestions for improvement, see the Hipparchus website:

https://www.hipparchus.org/
